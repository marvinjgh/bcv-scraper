# BCV Scraper

A simple web scraper to check the price of dollar in BCV

## How to use

```sh
# install dependence
npm install

# run
npm start
```

## Use as cli

```sh
# install global
npm i -g

bcv-scraper
```

## Terms of Use
- DO NOT use this scrapper for any other purpouse rather than getting the current exchange rate Bolivar-Dolar.
